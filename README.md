#  Trabalho 1 — Diferenciação Automática com Grafos Computacionais

##  Descrição

Este repositório contém a implementação de um sistema de **diferenciação automática** baseado em **grafos computacionais**.

O projeto tem como objetivo construir, do zero, um mecanismo capaz de:

* representar operações matemáticas como grafos computacionais
* propagar valores no grafo (forward pass)
* calcular gradientes automaticamente (backpropagation)
* manipular tensores multidimensionais e operações entre eles

Toda a implementação, explicações e experimentos estão documentados no notebook.
